# 後期課程進学要件チェックシート

**対象：ソーシャル・データサイエンス学部生**

後期課程判定時点で、以下の要件が全て満たされなければ後期課程に進学できません。

## 【総合】

- [ ] 1. 在学年数が通算で2年間以上であること。
- [ ] 2. 総修得単位数が54単位以上であること。

## 【ソーシャル・データサイエンス学部が開講する学部教育科目以外の科目】

- [ ] 3. 以下(1)～(3)に掲げる各科目について、それぞれの修得単位数が必要単位数以上であること。
  - [ ] (1) 英語コミュニケーションスキル科目（＝「PACEⅠ・Ⅱ」）の修得単位数が8単位以上であること。
  - [ ] (2)  外国語科目の修得単位数の合計が6単位以上であること。
  - [ ] (3) 数理・情報科目のうち「線形代数Ⅰ」「微分積分Ⅰ」「統計」「プログラミング基礎」「情報リテラシー」「AI入門」の修得単位数の合計が8単位以上であること。

## 【ソーシャル・データサイエンス学部が開講する学部教育科目の履修】

- [ ] 4. 学部導入科目のうち、「ソーシャル・データサイエンス入門Ⅰ」「ソーシャル・データサイエンス入門Ⅱ」「ソーシャル・データサイエンスの法と倫理」を全て修得していること。
- [ ] 5. 学部教育科目のうち、「ソーシャル・データサイエンス入門Ⅰ・Ⅱ」「ソーシャル・データサイエンスの法と倫理」以外の修得単位数が所定の単位以上であること。
  - [ ] (1) 「経営学・経済学系科目」の修得単位数が2単位以上であること。
  - [ ] (2) 「法学・政治学・その他の社会科学系科目」の修得単位数が2単位以上であること。
  - [ ] (3) 修得単位数の合計が8単位以上であること。
- [ ] 6. 学部基礎科目のうち、「回帰分析Ⅰ」「実践的機械学習Ⅰ」「プログラミングⅠ」の単位（合計6単位）を全て修得していること。
