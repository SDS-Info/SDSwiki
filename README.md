# SDS 履修ガイドブック

一橋大学ソーシャル・データサイエンス学部（SDS）の非公式履修ガイドブックの編集用リポジトリです。

## ファイル構成

| ファイル | 説明 |
|---------|------|
| `guide.md` | ガイドブック本文（Markdown） |
| `courses.csv` | 全科目データ（授業名・内容・開講日時・教員・評価基準・難易度・成績分布） |
| `courses.md` | 科目データの Markdown テーブル版 |
| `raw/` | 元データ（PDF・Word・CSV等） |

### `raw/` の中身

| ファイル | 説明 |
|---------|------|
| `SDS履修ガイドブック2025ver1.2.pdf` | 前年度版の完成 PDF |
| `履修ガイドブック草稿20260220.docx` | 2026年度版の草稿 |
| `courses_2026.csv` | 2026年度の科目データ（更新作業用） |
| `grades_2024.csv` | 2024年度の成績分布データ |

## 編集の流れ

1. `raw/` 内の元データ（科目一覧・成績分布等）をもとに `courses.csv` を更新
2. `guide.md` で本文テキストを編集
3. 最終的にデザインツールで PDF に組版
