## 実装済み機能

### 科目一覧ページ（courses.html）

- **キーワード検索**: 科目名・教員名・授業内容をリアルタイム検索（200ms デバウンス）
- **カテゴリフィルタ**: 4カテゴリ（SDS科目 / 社会科学 / データサイエンス / PACE）
- **科目区分フィルタ**: 5区分（学部導入 / 学部基礎 / 学部発展 / ゼミ関連 / 全学共通）
- **科目カード**: カテゴリバッジ、授業名、日時、教員、難易度（5段階ドット）、成績ミニバー
- **詳細モーダル**: クリックで開く詳細画面（評価基準、成績分布グラフ、難易度）
- **成績分布グラフ**: A+/A/B/C/F の横棒グラフ（色分け・凡例付き）

### 履修の基本情報ページ（rishu.html）

- **Markdown パーサ**: `guide.md` をクライアント側で HTML に変換
- **サイドバーナビ**: IntersectionObserver による現在位置追跡＋スムーズスクロール
- **レスポンシブ対応**: モバイルではハンバーガーメニューに切り替え

### 共通機能

- **レスポンシブデザイン**: 768px ブレークポイント、モバイルファースト
- **アニメーション**: カードのスタッガード表示、ホバーエフェクト
- **アクセシビリティ**: セマンティック HTML、ARIA ラベル、キーボード操作対応

## 技術構成

```
docs/
├── index.html          トップページ
├── courses.html        科目一覧ページ
├── rishu.html          履修の基本情報ページ
├── app.js              全ページ共通スクリプト（543行）
├── style.css           全ページ共通スタイル（1,138行）
├── images/
│   └── logo-sds.png    ロゴ画像
├── courses.csv         → ../courses.csv（シンボリックリンク）
├── guide.md            → ../guide.md（シンボリックリンク）
└── .nojekyll           Jekyll無効化
```

### デザインシステム

- **カラー**: CSS Variables で管理。ゴールド系プライマリ（#d4c82b）、カテゴリ別アクセントカラー
- **フォント**: Noto Sans JP（400/500/700/800）
- **レイアウト**: CSS Grid + Flexbox、最大幅 1260px
- **エフェクト**: glassmorphism（トップバー）、ホバー浮上、スタッガードアニメーション

## デプロイ方法

```
1. courses.csv / guide.md をルートで編集
2. git push → GitHub Pages に自動反映（ビルド不要）
```

シンボリックリンクにより `docs/` 内のデータは自動同期。HTML/CSS/JS の変更も push のみで反映。
